Проект написать на ASP.NET (Net 9)

Framework:
1. Entity Framework (PostgreSQL)
2. BCrypt Core (хэширование пароля)

Страницы:
1. ../registration - страница для регистрации пользователя
2. ../authorization - страница для авторизации пользователя
3. ../account - после авторизации или регистрации автоматически переносит на страницу с данными пользователя (особой функции не несет, только для подтверждения входа на аккаунт)
